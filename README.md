## Introduction
This app is a project you can add some your notes and manage them. Developed by HTML, CSS and Javascript. In this project, the object-oriented programming paradigm is used so that each layer performs its task and everything is more organized. Notes are also stored in the browser's local storage. This app has been implemented in a responsive design. You can see live in this [link](https://note-app-sigma-bice.vercel.app) .
## Features
1. Add note
2. Select note to view its information
3. Edit note with any change on the title input and body input
4. Delete note
5. Sync of notes with the browser's local storage
